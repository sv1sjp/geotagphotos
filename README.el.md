# GeoTag Photos

> Προσθέστε συντεταγμένες GPS στις JPEG φωτογραφίες σας απευθείας από την εφαρμογή Αρχεία του Nextcloud.

Το **GeoTag Photos** σας επιτρέπει να γράφετε μεταδεδομένα GPS EXIF σε αρχεία JPEG, ώστε να εμφανίζονται στον χάρτη στο [Nextcloud Maps](https://apps.nextcloud.com/apps/maps) και το [Nextcloud Memories](https://apps.nextcloud.com/apps/memories) μετά από επανασάρωση — χωρίς ανάγκη για λογισμικό desktop!

**Δημιουργός:** Dimitris Vagiakakos ([@sv1sjp](https://github.com/sv1sjp)) — [tuxhouse.eu](https://www.tuxhouse.eu)  
**Άδεια:** GNU AGPL v3 ή νεότερη  
**Nextcloud:** 33

## Χαρακτηριστικά

- **Προσθήκη / Αντικατάσταση GPS** — γράψτε γεωγραφικό πλάτος, μήκος και προαιρετικό υψόμετρο σε οποιοδήποτε JPEG
- **Εμφάνιση πριν την αντικατάσταση** — οι υπάρχουσες συντεταγμένες εμφανίζονται στο παράθυρο πριν τις αλλάξετε
- **Εκκαθάριση GPS** — αφαίρεση όλων των ετικετών GPS EXIF από μια φωτογραφία με ένα κλικ
- **Μαζική προσθήκη** — επιλέξτε πολλά JPEG και εφαρμόστε τις ίδιες συντεταγμένες σε όλα ταυτόχρονα
- **Επικόλληση από Πρόχειρο** — επικολλήστε συντεταγμένες απευθείας από το Google Maps ή οποιαδήποτε εφαρμογή GPS· υποστηρίζει δεκαδικές μοίρες (`38.083454, 23.697252`) και DMS (`38°05'21.1"N 23°42'42.0"E`)
- **Ασφαλής εγγραφή EXIF** — αγγίζονται μόνο οι ετικέτες GPS· τα `DateTimeOriginal`, προσανατολισμός, πεδία κάμερας και όλα τα άλλα μεταδεδομένα δεν τροποποιούνται ποτέ

## Απαιτήσεις

| Συστατικό | Έκδοση |
|---|---|
| Nextcloud | 33 |
| PHP | 8.1 ή νεότερη |
| **exiftool** | οποιαδήποτε πρόσφατη έκδοση |

### Εγκατάσταση exiftool στον διακομιστή

```bash
# Debian / Ubuntu
apt install libimage-exiftool-perl

# Fedora / RHEL / CentOS
dnf install perl-Image-ExifTool

# Alpine Linux
apk add exiftool

# Arch / Manjaro
pacman -S perl-image-exiftool
```

> **Σημείωση:** Το exiftool πρέπει να είναι εγκατεστημένο στο μηχάνημα που εκτελεί τη διαδικασία PHP του Nextcloud (μέσα στο container αν χρησιμοποιείτε Docker/Podman).

## Εγκατάσταση

### Από το Nextcloud App Store *(προτεινόμενο - σύντομα)*

Αναζητήστε το **GeoTag Photos** στις **Ρυθμίσεις → Εφαρμογές** και κάντε κλικ στο Εγκατάσταση.

### Από αρχείο έκδοσης

1. Κατεβάστε το `geotagphotos.tar.gz` από τη [σελίδα εκδόσεων](https://github.com/sv1sjp/geotagphotos/releases).
2. Αποσυμπιέστε στον φάκελο εφαρμογών του Nextcloud:
   ```bash
   tar -xzf geotagphotos.tar.gz -C /path/to/nextcloud/apps/
   ```
3. Ενεργοποιήστε την εφαρμογή:
   ```bash
   occ app:enable geotagphotos
   ```

### Από πηγαίο κώδικα

```bash
cd /path/to/nextcloud/apps
git clone https://github.com/sv1sjp/geotagphotos.git
cd geotagphotos
npm install
npm run build
occ app:enable geotagphotos
```

## Χρήση

### Μία φωτογραφία

1. Κάντε δεξί κλικ σε οποιοδήποτε JPEG στην εφαρμογή Αρχεία (ή χρησιμοποιήστε το μενού `⋮`).
2. Κάντε κλικ στο **Προσθήκη Ετικέτας Γεωτοποθεσίας**.
3. Το παράθυρο εμφανίζει τυχόν υπάρχουσες συντεταγμένες. Εισάγετε νέες και κάντε κλικ στο **Αποθήκευση** (ή **Αντικατάσταση** αν υπάρχουν ήδη GPS δεδομένα).
4. Για αφαίρεση όλων των GPS δεδομένων, κάντε κλικ στο **Εκκαθάριση GPS**.

### Πολλές φωτογραφίες

1. Επιλέξτε αρκετά JPEG χρησιμοποιώντας τα πλαίσια ελέγχου.
2. Κάντε κλικ στο **Προσθήκη Ετικέτας Γεωτοποθεσίας** στη γραμμή εργαλείων επιλογής.
3. Εισάγετε τις συντεταγμένες που θα εφαρμοστούν σε όλες τις επιλεγμένες φωτογραφίες και κάντε κλικ στο **Αποθήκευση**.

### Επικόλληση από Πρόχειρο

Κάντε κλικ στο **Επικόλληση από Πρόχειρο** μέσα στο παράθυρο για αυτόματη συμπλήρωση των πεδίων γεωγραφικού πλάτους και μήκους. Υποστηριζόμενες μορφές:

| Μορφή | Παράδειγμα |
|---|---|
| Δεκαδικές μοίρες | `38.083454, 23.697252` |
| Δεκαδικές μοίρες (με πρόσημο) | `-33.8688, -70.6693` |
| DMS | `38°05'21.1"N 23°42'42.0"E` |

Το επιπλέον κείμενο ή κενές γραμμές στο πρόχειρο (π.χ. κατά την αντιγραφή από Google Maps) αγνοούνται αυτόματα.

## Μορφή συντεταγμένων GPS

Η εφαρμογή χρησιμοποιεί **δεκαδικές μοίρες (DD)** — την ίδια μορφή που χρησιμοποιεί το Google Maps.

| Κατεύθυνση | Πρόσημο | Παράδειγμα |
|---|---|---|
| Βορράς | + (θετικό) | `37.9838` |
| Νότος | − (αρνητικό) | `-33.8688` |
| Ανατολή | + (θετικό) | `23.7275` |
| Δύση | − (αρνητικό) | `-74.0060` |

Το υψόμετρο είναι σε **μέτρα**, θετικό πάνω από τη στάθμη της θάλασσας και αρνητικό κάτω.

## Ετικέτες EXIF που γράφονται

Γράφονται μόνο αυτές οι ετικέτες — τίποτε άλλο δεν αγγίζεται:

| Ετικέτα | Περιγραφή |
|---|---|
| `GPSLatitude` | Απόλυτη τιμή γεωγραφικού πλάτους |
| `GPSLatitudeRef` | `N` (βορράς) ή `S` (νότος) |
| `GPSLongitude` | Απόλυτη τιμή γεωγραφικού μήκους |
| `GPSLongitudeRef` | `E` (ανατολή) ή `W` (δύση) |
| `GPSAltitude` | Υψόμετρο σε μέτρα *(μόνο αν παρέχεται)* |
| `GPSAltitudeRef` | `0` πάνω από τη στάθμη της θάλασσας / `1` κάτω |

## Ασφάλεια

- **Αυθεντικοποίηση** — όλες οι διαδρομές API απαιτούν συνδεδεμένο χρήστη (`@NoAdminRequired`).
- **Εξουσιοδότηση** — τα αρχεία επιλύονται μέσω `IRootFolder→getUserFolder()`, οπότε οι χρήστες μπορούν να επεξεργαστούν μόνο αρχεία που τους ανήκουν ή έχουν πρόσβαση εγγραφής.
- **Έλεγχος MIME** — αρχεία που δεν είναι JPEG απορρίπτονται στο επίπεδο PHP ακόμα και αν παρακαμφθεί το φίλτρο του frontend.
- **Command injection** — το `ExifService` καλεί το exiftool μέσω `proc_open` με εντολή σε μορφή **πίνακα** (όχι συμβολοσειρά shell), καθιστώντας την injection δομικά αδύνατη.
- **Path traversal** — οι διαδρομές αρχείων επιλύονται μέσω του στρώματος αποθήκευσης του Nextcloud και επαληθεύονται με `realpath()` πριν μεταβιβαστούν στο exiftool.

## Αντιμετώπιση προβλημάτων

**«exiftool is not installed»**  
Εγκαταστήστε το exiftool στον διακομιστή — δείτε [Απαιτήσεις](#απαιτήσεις). Αν χρησιμοποιείτε container, εγκαταστήστε το μέσα στην εικόνα container.

**«Only local storage is supported»**  
Η εξωτερική αποθήκευση (S3, WebDAV, SFTP κ.λπ.) δεν υποστηρίζεται. Η φωτογραφία πρέπει να βρίσκεται στον τοπικό δίσκο του διακομιστή Nextcloud.

**Η φωτογραφία δεν εμφανίζεται στους Χάρτες μετά την προσθήκη ετικέτας**  
Εκτελέστε `occ maps:scan-photos` για να αναγκάσετε τους Χάρτες να ανευρετηριάσουν εκ νέου τα GPS δεδομένα από την κρυφή μνήμη αρχείων.

**Οι συντεταγμένες φαίνονται λανθασμένες (αντεστραμμένο πρόσημο)**  
Ελέγξτε το πρόσημο: τα γεωγραφικά πλάτη νότια του ισημερινού και τα γεωγραφικά μήκη δυτικά του μεσημβρινού Greenwich πρέπει να είναι **αρνητικά**.
